# 🏷️ Tag Production Dashboard

מערכת ניתוח ויזואלית לבדיקת תגיות (Tags) על בסיס קבצי Excel.  
האפליקציה מאפשרת להעלות קובץ נתונים של מסד הנתונים וקובץ תגיות, לבצע בדיקה ביניהם,  
ולנתח את פילוח התגים לפי שנת הייצור – כולל הצגה בגרף עוגה אינטראקטיבי וייצוא לקובץ Excel.

---

## 🚀 תכונות עיקריות

- 📂 העלאת קובץ **Database** (Excel)
- 🔍 בדיקת קובץ **Tags** והשוואה מול המסד
- 🥧 הצגת **פילוח שנתי** של תגים בגרף עוגה אינטראקטיבי
- ⬇️ הורדת הגרף כקובץ **PNG** עם רקע לבן
- 📊 ייצוא תוצאות הבדיקה לקובץ **Excel**
- 🎨 ממשק משתמש ברור, צבעוני ונגיש
- ✅ לחצנים מעוצבים וסטטוס פעולה ברור

---

## 🛠️ טכנולוגיות בשימוש

| טכנולוגיה | שימוש |
|------------|--------|
| **React.js** | ממשק המשתמש |
| **Chart.js + chartjs-plugin-datalabels** | יצירת גרף עוגה אינטראקטיבי |
| **Axios** | תקשורת עם השרת |
| **FileSaver.js** | הורדת קבצים בצד הלקוח |
| **XLSX.js** | קריאה וכתיבה של קובצי Excel |
| **Node.js / Express** | צד שרת (אם קיים) |
